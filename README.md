OneStack Gen 🛠️✨

 🌐 Build Apps with Just a Prompt

OneStack Gen is an AI-powered full stack web application that transforms your ideas into structured plans for websites, dashboards, apps, and internal tools — instantly using just a text prompt!


📌 Project Description

Ever wanted to build something but didn't know where to start? OneStack Gen solves that!

Just type what you want to build — like `"fitness app"` or `"sales dashboard"` — and OneStack Gen instantly generates a complete structured plan for you. The result can be downloaded and used as a blueprint for your next project.

The system processes prompts using a **Flask REST API**, generates intelligent plans using a **smart keyword engine**, saves everything to a **SQLite database**, and displays results beautifully on the frontend.



 🚀 Features

- 🌐 **Website Generator** — Generate website plans from a simple prompt
- 📊 **Dashboard Generator** — Create dashboard layouts instantly
- 📱 **App Generator** — Build app structures with one click
- 🛠 **Internal Tool Builder** — Generate internal tool plans instantly
- 📜 **History Page** — View all previously generated results
- ⬇️ **Download Results** — Save your generated plan as a `.txt` file
- 🔐 **Login / Signup** — User authentication system
- 🧠 **Smart Generator** — Understands 20+ keywords like `hospital`, `crm`, `inventory`, `fitness` and more!
